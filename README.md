# Sales Performance and Forecasting Dashboard

This project is an interactive Power BI dashboard designed to analyze sales performance and forecast future sales using historical data.

It includes data preparation, KPI analysis, and built-in time series forecasting, all visualized in a clear and user-friendly dashboard.

## Features
- Sales performance analysis
- KPI tracking (Revenue, Margin %, Sales Growth)
- Sales analysis by product, region, and time
- Time series forecasting using Power BI
- Interactive filters and dynamic visuals

## Tools
- Power BI
- Python (ETL)
- CSV datasets
